# Sunergy <sup>&copy;</sup><small>1.0.0</small>

> 跨平台深度学习框架。

* 纯C语言编写
* 通过给定的结构定义文件和结构权重文件实现各种深度模型
* 针对Linux、Windows、iOS、Android的优化
* 针对GPU和CPU的优化
* 提供C/C++和Python接口
