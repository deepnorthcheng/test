## Sunergy

> 跨平台深度学习框架。

## 是什么

Sunergy是一个跨平台深度学习框架。它由纯C语言编写，所以天生就能够跨平台编译。Sunergy的灵活设计架构使它能够通过给定不同的结构定义文件和结构权重文件而实现不同的深度模型。Sunergy也提供了很多语言的接口。请参考[入门](/zh-cn/quickstart.md)了解更多。

## 特性

* 纯C语言编写
* 通过给定的结构定义文件和结构权重文件实现各种深度模型
* 针对Linux、Windows、iOS、Android的优化
* 针对GPU和CPU的优化
* 提供C/C++和Python接口
